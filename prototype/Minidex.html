Minidex [2022]

This project is a a web scraping tool in Python to extract Pokémon data from the French website Poképédia.

The tool gathers information about Pokémon from all generations, making it easy to consult their details in a structured format. It’s a handy resource for exploring comprehensive Pokémon information efficiently.
